Пример страницы версии html 4.01 (strict)

Этот файл подготовлен в соответствии с версией html 4.01 (strict).

У него задан doctype следующего вида:

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

Это достаточно строгий тип документа, не допускающий отклонений от стандарта
