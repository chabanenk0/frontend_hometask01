Пример страницы версии xhtml 1.0 (transitional)

Этот файл подготовлен в соответствии с версией xhtml 1.0 (transitional).

У него задан doctype следующего вида:

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

Transitional doctype не такой строгий, он используется для документов, которые по каким-то причинам сложно привести к требованиям данной версии html, допускается его использование только в самом крайнем случае.

Список тегов и атрибутов, а также особенности, которые не разрешены в strict, но разрешены transitional, чаще всего связаны с представлением документа (выравнивание, цвет, вид шрифта, см. полный список). Принято все, что относится к содержанию (контенту), оформлять в html, а оформление - выносить в css). В этом документе не соблюдены некоторые требования строгости. Попробуйте найти их.
