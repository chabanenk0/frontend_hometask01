Теги в заголовке html

В секции head документа html используются такие теги:

  1. title - для задания заголовка документа (отображается в заголовке окна)
  2. meta - для определение кодировки текста и другой мета-информации
  3. link - для подключения css-стилей, возможно еще чего-то
  4. base - для задания базового url для сайта
  5. script - для подключения внешних javascript либо определения скриптов в средине документа
  6. style - для определения стилей в средине документа
  7. object - ??
